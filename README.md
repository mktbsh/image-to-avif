# img2avif

## CLI

`single directory`:

```sh
img2avif ./target-dir
```

`multiple directories`:

```sh
img2avif ./dir1 ./dir2 ./dir3
```

`with quality`:

```sh
img2avif ./target-dir --q 50
```

default quality is 60

`with remove target dir`:

```sh
img2avif ./target-dir --rmdir
```

`no convert to avif`:

```sh
img2avif ./target-dir --original
```
